# React-Boilerplate

## 支持
- [x] react-router v4
- [x] webpack3
- [x] eslint
- [x] prettier
- [ ] webpack插件整理

## 目录说明
- dist: 打包后的文件目录
- public: 开发时DLL文件(对package.json中的依赖生成DLL以优化编译性能)目录
- src: 项目代码目录
- entry: index.html模版文件
- webpack: webpack的plugin和module目录,以避免重复定义

## 运行
- npm install
- npm run local.dll(dependencies有变动就运行一次)
- npm run local.dev# react-boilerplate
